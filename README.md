# Telegram AI Bot (MashaGPT)

Бот для генерации текста, изображений, видео, аудио и анимации через API MashaGPT.

## Возможности

- **Текст** – бесплатно, без лимита (GPT-4o-mini, GPT-5-mini, DeepSeek и др.)
- **Изображения** – бесплатно, 5 в неделю (Midjourney, Flux, Nano Banana, Z-Image и др.)
- **Видео** – платно (Sora, Kling, Wan, Veo и др.)
- **Обработка изображений** – платно (удаление фона, апскейл, замена лица)
- **Аудио** – бесплатно (озвучка, диалоги) / платно (звуковые эффекты)
- **Аватар и анимация** – платно
- **Баланс** – пополнение через Telegram Stars (100 звёзд = 100 промтов)
